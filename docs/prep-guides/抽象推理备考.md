## 各章节详细教学内容

### 第1章：How to ACE Abstract Reasoning Tests（总体策略）

**抽象推理的定义：**
以图像为载体的推理，与其他推理题不同，因为图像包含多个维度的属性：
- 形状变换（Transformation of shapes）
- 方向与空间位置（Direction and position）
- 颜色（Colours）
- 大小（Size）
- 叠加（Superimposition）
- 角度变换（Angle transformation）
- 数字规律（Number patterns）

所有这些属性都需要在"找到存在的规律"这一目标下进行评估。

**核心能力：规律识别（Pattern Detection）**
> 记住：规律就在那里，你的工作是找到它。

**每道题的2大步骤：**
1. **尝试找到规律**（Try to find the pattern）
2. **选择符合规律的答案**（Select the answer that fits）

**时间策略：**
- 快速扫题：能找到规律则解，找不到则跳过
- 剩余时间末尾：猜（B或C概率稍高）
- 不要在单题上超过计划时间

**课程覆盖的规律类型：**
方向 → 位置 → 旋转 → 大小 → 颜色 → 角度 → 叠加 → 变换 → 反转 → 翻转 → 堆叠 → 排序 → 字母 → 数字

---

### 第2章：Critical Test Strategies（关键解题技巧）

**3大核心解题策略：**

1. **每个图像由各部分组成（Made up of parts）**
   - 把整体图像分解为各个独立部分来分析
   - 例：整幅南瓜点画图 → 单个点的颜色规律

2. **规律可以存在于单张图像内或图像序列中（Patterns within image OR sequence）**
   - 序列（Sequence）：一系列图像中的变化规律
   - 不要只看单张图，要在整个序列中验证规律

3. **共性与孤立性（Commonality and Isolation）**
   - **共性**：找出所有图像中共同的属性（贯穿整个序列的规律）
   - **孤立性**：找出只有某张图像特有的属性（可能是异类，也可能是规律的关键）
   - 在后续章节中会反复应用这两个概念

**解题流程：**
1. 把图像分解成部分
2. 在单张图内找规律
3. 在序列中找规律
4. 用共性与孤立性交叉验证
5. 选择符合规律的答案

---

### 第3章：Direction（方向规律）

**核心概念：把方向想象成罗盘（Compass）**

**方向规律的特点：**
- 图像的某部分按方向移动/旋转
- 常见方向：顺时针（clockwise）、逆时针（anti-clockwise）
- 常见角度：45° 和 90°
- 8个罗盘方向：北（N）、南（S）、东（E）、西（W）、东北（NE）、东南（SE）、西南（SW）、西北（NW）

**识别方向规律的关键问题：**
- 某个元素面朝哪个方向？
- 对立方向是什么？（N↔S，E↔W，NE↔SW）
- 每步移动了多少度？

**关键规则：**
> 用罗盘思考方向；顺时针/逆时针；45°和90°最常见；识别起始方向和步进角度。

---

### 第4章：Position（位置规律）

**核心概念：把位置想象成地图（Map）**

**位置与方向的区别：**
- 方向 = 罗盘（指向哪个方向）
- 位置 = 地图（在哪个具体位置）

**常见位置规律类型：**
- 紧邻/相邻（adjacent）另一个图像
- 在边或角上，然后移到对面的边或角
- 相对于另一个元素保持固定位置（另一元素移动时位置不变）
- 相对于另一元素变化的相对位置
- 比较位置：是否紧挨着对比色或对比尺寸的元素？

**分析位置时需问自己：**
- 某元素位于哪里？
- 它的位置是如何移动的（左→右，上→下）？
- 它相对于其他元素的位置关系是什么？

**关键规则：**
> 用地图坐标思考位置；关注左右/上下/对角线方向的移动；留意与其他元素的相对位置。

---

### 第5章：Rotation（旋转规律）

**旋转的三要素：**
1. **旋转方向**：顺时针或逆时针
2. **旋转角度**：45°、90°、180° 最常见
3. **旋转中心**：形状围绕哪个点旋转

**旋转中心的常见类型：**
- 垂直轴旋转
- 水平轴旋转
- 斜角旋转
- 页面上某固定点旋转（最常见于2D图形题）

**旋转与方向的关系：**
旋转常与方向和位置规律结合使用——例如：
- 形状在正方形的4个角上，每次顺时针旋转90°

**识别旋转规律的关键问题：**
- 从起始点旋转了多少度？
- 是顺时针还是逆时针？
- 每次旋转的角度是否一致？

**关键规则：**
> 明确旋转三要素（方向/角度/中心）；90°和45°最常见；旋转常与方向/位置组合出现。

---

### 第6章：Size（大小变化规律）

**大小规律的类型：**
- 整体图像增大或减小
- 特定部分增大或减小（如某条边长度变化）
- 一个元素相对于另一个元素在大小上增减（如小正方形在变大的大正方形内）
- 两种或多种大小变化同时发生（如一个矩形水平增大，另一个垂直减小）
- 大小变化有方向性（垂直/水平/对角线方向增大）
- 大小按特定单位增加（如每次增加1格）

**大小规律的特点：**
- 几乎总是与方向和位置规律共存
- 需要注意大小变化的方向和幅度

**关键规则：**
> 识别增大/减小趋势；注意变化方向（水平/垂直/对角）；找出每步增减的单位量。

---

### 第7章：Colours（颜色/填充规律）

**颜色规律的三种类型：**
1. **有无（Absence and presence）**：黑→白→黑（交替出现）
2. **色调渐变（Shades）**：黑→灰→白，循环重复
3. **填充样式（Patterns within colour）**：点状→空白→点状

**常见颜色规律示例：**
- 每行/每列至少包含1个白色、1个黑色、1个灰色方块
- 对角线条纹填充
- 网格状黑白交替

**颜色规律的进阶：**
- 颜色出现在图像的特定部分（局部颜色变化）
- 颜色规律与方向/位置规律组合使用

**关键规则：**
> 观察颜色的有无、渐变和填充样式；颜色可以是局部的；常与其他规律组合。

---

### 第8章：Angles（角度变化规律）

**角度的基本概念：**
- 角度 = 倾斜程度（the slant of something）
- 完整圆 = 360°，角度是圆的一个分数部分
- 锐角 < 90° < 钝角 < 180° < 优角 < 360°

**角度在抽象推理中的三种出现方式：**
1. **形状内部的角度**：形状属性的一部分（如平行四边形→菱形，角度变化）
2. **方向与位置中的角度**：元素在正方形4个角上按90°顺/逆时针移动
3. **角度本身的变化规律**（本章新内容）：
   - 角度递增
   - 角度递减
   - 角度重复（固定角度交替出现）

**角度规律类型：**
- 内角（Inside angles）：形状内部角度的变化
- 外角（Outside angles）：形状之外的空间角度变化

**关键规则：**
> 角度可以是形状属性、方向工具或独立变化的规律；识别递增/递减/重复三种模式。

---

### 第9章：Superimposition（图形叠加规律）

**叠加的定义：**
将一个图像放在另一个已有图像上方，可以增强整体效果，也可以隐藏某些内容。

**在抽象推理中的表现：**
1. 先画出图像的一部分（看起来像独立图像）
2. 再画出图像的另一部分
3. 将两部分叠加后才能看出完整图像

**叠加规律的两个方向：**
- **增加（Addition）**：一个图像叠加在另一个上面
- **减少（Reduction）**：从现有图像中减去一个图像

**识别叠加规律的时机：**
当方向、位置、旋转、颜色、大小等常见规律都已排除后，考虑叠加规律。

**叠加的变体：**
- 简单叠加（直接覆盖）
- 透明叠加（可见底层）
- 部分叠加（交叉区域）
- 减法叠加（去除重叠部分）

**关键规则：**
> 叠加 = 图像A + 图像B = 结果图；或结果图 - 图像A = 图像B；识别各部分再合并验证。

### 第10章：Transformations（图形变换综合）

**形状变换的分析框架：**
把每个形状看作由以下部分组成：
- **角度（Angles）**：有几个角，角度是多少
- **边（Sides）**：有几条边，边长如何
- **位置（Position）**：处于什么位置/朝向
- **大小（Size）**：是否是常规大小

**四边形的6种形态变换：**
平行四边形 ↔ 长方形 ↔ 正方形 ↔ 菱形（diamond）↔ 梯形 ↔ 风筝形（kite）

**常见变换问题：**
- 平行四边形→风筝形：哪些属性改变了？
- 长方形→正方形：所有边等长化
- 菱形→风筝形：对称轴改变

**核心思想：**
> 将图形分解成各组成部分，观察哪个部分发生了变化，就是变换的规律所在。

### 第11章：Tricky Patterns & "Odd One Out"（复杂规律与找异类题）

**复杂题型（Tricky Questions）的特点：**
- 将多种规律组合在一起
- 故意设计为看起来简单，但实际包含需要精确捕捉的细节
- 使用不寻常的排列（如轮状、方块组合）

**常见错误原因：**
- 错误识别规律类型（看似形状变换，实为位置变化）
- 把整个形状看作一体而非各部分
- 多个可能规律的干扰

**"找异类"（Odd One Out）题型策略：**
> 反向思维：找出不符合其他图像所有规律的那一个

**找异类的步骤：**
1. 找出所有图像的共同规律
2. 逐一检查每个图像是否符合所有共同规律
3. 找出不满足某个共同规律的图像
4. 验证：其他所有图像都满足该规律，而它不满足

**关键规则：**
> 复杂题先分解再重组；找异类用排除法，找所有图像的共性再找例外。

### 第12章：Inverse（反转/取反规律）

**反转的定义：**
取某物的对立面/反面，与颜色规律容易混淆，但区别在于：
- **颜色规律**：在序列中按顺序变化（黑→白→黑）
- **反转规律**：1对1的关系，一张图是另一张图的反面

**反转的表现形式：**
- 前景与背景互换（黑底白图 ↔ 白底黑图）
- 包含关系互换（圆包含在正方形内 ↔ 正方形包含在圆内）

**识别方法：**
- 寻找1对1关系
- 找到"互为相反"的图像对

**解题方法：**
找到对立面——如"前景图"的反转就是"背景图"，就像交换位置一样。

**关键规则：**
> 反转 = 1对1的对立关系；前景↔背景；内包含↔外包含；不要与序列性颜色变化混淆。

### 第13章：Flips（翻转规律）

**翻转（Flip）= 反射（Reflection）**

**翻转的三要素：**
1. 不可见的轴（可以是垂直轴、水平轴或斜轴）
2. 原始图像
3. 翻转后得到镜像

**关键翻转规则：**
- **水平翻转（沿垂直轴）**：原图左边的内容 → 镜像右边；原图右边 → 镜像左边
- **垂直翻转（沿水平轴）**：原图上方的内容 → 镜像下方；原图下方 → 镜像上方

**翻转特性：**
- 镜像与原图大小和尺寸完全相同
- 翻转后形状本身不变，只是方向改变

**与旋转的区别：**
- 旋转180°的图像看起来像翻转，但实际不同（不对称图形时差异明显）

**关键规则：**
> 翻转需要轴；水平翻转=左右互换；垂直翻转=上下互换；大小尺寸不变。

---

### 第14章：Stacks（堆叠规律）

**堆叠 = 翻转的延伸**

堆叠在翻转的基础上，测试图像由多个**层（layers）**组成时的翻转效果。

**堆叠翻转的关键规则：**
- 翻转后，**原底层变为最顶层**，成为最可见的层
- 如果底层是不透明的（opaque），翻转后它会遮盖所有其他层

**理解堆叠顺序：**
翻转前：层1（顶）/ 层2 / 层3（底）
翻转后：层3（顶，最可见）/ 层2 / 层1（底）

**组合效果：**
翻转轴（水平/垂直/斜角）× 层的顺序变化 = 最终结果

**关键规则：**
> 堆叠 = 多层图像翻转；注意翻转后层级顺序逆转；底层变顶层，顶层变底层。

---

### 第15章：Ordering（排序规律）

**排序题的特点：**
要求将一组图像（通常是答案选项）按逻辑视觉顺序排列，并选出第1、2、3、4或5位的图像。

**两种主要排序规律：**

1. **峰形（先增后减）**
   - 图像从小到最大再到小
   - 第一张和最后一张相似（形成首尾对称）
   - 中间图像是"峰值"

2. **单向增减（小到大 或 大到小）**
   - 存在一个"最大值"图像和一个"最小值"图像
   - 中间图像是"半完成"状态

**根据题目要求选择策略：**
- 找**中间**图像 → 找两个相似的端点图像，中间是过渡态
- 找**第一张**图像 → 找"最小/最初"状态
- 找**最后一张**图像 → 找"最大/最终"状态

**关键规则：**
> 识别是"峰形"还是"单向"规律；根据题目要求（中间/第一/最后）选择对应策略。

### 第16章：Letters in Patterns（图形中的字母规律）

**字母在抽象推理中的角色：**
由于抽象推理是非语言的，字母出现时，它们代表**图像或图像的一部分**，就像符号一样。

**字母的两种类型：**
1. **公共字母（Common Letter）**：重复出现的字母，代表基础图像
2. **修饰字母（Modifier Letter）**：改变公共字母图像外观的字母

**解题步骤：**
1. 找出重复出现的公共字母
2. 找出与公共字母相关联的重复图像
3. 将字母与图像对应（建立字母→图像的映射）
4. 找出修饰字母
5. 观察修饰字母如何改变基础图像
6. 建立规律描述，用于找出缺失图像

**例：** 字母A代表圆形，字母B代表"将圆变成正方形"，则AB = 正方形

**关键规则：**
> 字母 = 图像符号；区分公共字母与修饰字母；建立映射后找规律。

### 第17章：Numbers in Patterns（图形中的数字规律）

**数字规律的表现形式：**
数字通常以视觉符号表示，例如：
- 圆点（dots）的数量
- 线段（lines）的数量

**识别数字规律的3个信号：**
1. 图像中包含数量性元素（点、线、形状的个数）
2. 没有明显的视觉规律（如无重复、无方向变化）
3. 数量本身呈现规律（递增/递减/组合）

**数字规律类型：**
- 加法/乘法递增
- 减法/除法递减
- 加减组合

**解题方法：**
> 当看不出视觉规律时，转换思维——**把图像中的元素数量化**，用数字推理方法分析

**与数字推理课程的联系：**
本章节是抽象推理与数字推理的交叉点，如果识别到数量模式，直接套用数列规律方法。

**关键规则：**
> 看到点/线的数量时，转换为数字推理思维；加/减/乘/除找规律。

### 第18章：Revision & Final Exam（复习与期末考试）

**各章节核心记忆点：**

| 章节 | 关键原则 |
|------|---------|
| 关键策略 | 图像由各部分组成；分解后更容易找规律 |
| 方向 | N/S/E/W等8个罗盘方向；识别面朝方向和对立方向 |
| 位置 | 某元素在哪里？如何移动（左右/上下）？ |
| 旋转 | 注意度数；从起始点变化了多少；顺/逆时针；常见90°和45° |
| 大小 | 增大/减小；有无重复 |
| 颜色 | 黑/白/灰；填充样式；注意局部颜色变化 |
| 角度 | 角度递增/递减/重复；注意形状内角和方向移动的角度 |
| 叠加 | 两个图像叠加；识别各层再合并 |
| 变换 | 形状由部分组成；哪个部分发生了变化 |
| 复杂/找异类 | 多规律组合；找异类用共性+例外法 |
| 反转 | 1对1对立关系；前景↔背景 |
| 翻转 | 水平翻转=左右互换；垂直翻转=上下互换 |
| 堆叠 | 翻转后底层变顶层 |
| 排序 | 峰形 or 单向；根据题目要求选策略 |
| 字母规律 | 公共字母+修饰字母；建立映射 |
| 数字规律 | 数量→数字推理思维 |

**总体应试策略：**
> 识别规律类型 → 快速能解则解 → 否则跳过 → 最后猜

- 标准版：30题，限时约25分钟
- 进阶版：50题，限时30分钟
