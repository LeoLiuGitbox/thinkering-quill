### 第1章：How to ACE Competitive Maths Tests - Grade 7 Entry（总体策略）

**竞争性数学考试与学校数学的区别：**
- 测试年7级课程水平（甚至更高）的数学知识
- 侧重**数学推理和应用**（解题建模），而非机械计算

**核心能力：**
1. **课程知识（Curriculum Knowledge）**：达到7年级数学水平
2. **阅读与建模（Read & Formulate）**：从文字题中提取数学模型
3. **问题解决（Problem Solving）**：找到"缺失的桥"（missing link）

**每道数学题的2大步骤：**
1. 清晰理解题目要求 + 建立公式/计算方法
2. 解题计算

> 竞争性考试的难点在于步骤1（批判性思维和建模），而非步骤2（计算本身）。

**考试流程（6步法）：**
1. 考前备好知识
2. 读题后判断能否在规定时间内完成
3. 找到关键信息
4. 建立公式
5. 解题
6. 选择答案

**时间策略：**
- 快速扫题：能做则做，不确定则跳过
- 剩余30秒：能做则做，否则猜（B或C选项概率较高）
- 不要在单题上超过计划时间，每浪费的秒数等于放弃2-4道其他题

**常见错误：**
- 在一道题上花太长时间
- 误读题目
- 没有策略或方法

### 第2章：Maths Shortcuts & Tips（数学捷径与核心概念）

**数字类型知识：**
- **偶数/奇数规则**（用于验算答案或排除选项）：
  - 偶数 ±/× 偶数 = 偶数
  - 偶数 ± 奇数 = 奇数
  - 奇数 ± 奇数 = 偶数
  - 偶数 × 奇数 = 偶数
  - 奇数 × 奇数 = 奇数
- **质数（Prime）**：大于1且只能被1和自身整除
- **连续数（Consecutive）**：按顺序排列的数
- **因数（Factors）**：6的因数是1, 2, 3, 6
- **倍数（Multiples）**：6的倍数是12, 18, 24…

**心算捷径：**
- **大数加减（取整法）**：
  - 745 + 852 → 750 + 850 = 1600 → -5+2 = 1597
- **基础减法知识**：
  - 7+3=10 → 10-3=7, 10-7=3（以此类推）
- 数字1-10都能组合成对：6+4, 8+2, 1+9, 5+5…

**记忆要点：**
> 掌握质数、连续数、因数、倍数术语；大数运算用取整再调整；奇偶规则帮助验证/排除选项。

### 第3章：The Number Line and The Four Operators（数轴与四则运算）

**数轴（Number Line）：**
- 正数在0右侧，负数在0左侧
- 负数运算规则：负负得正，负正得负
- 数轴可视化帮助理解加减运算方向

**四则运算要点：**
- **加法**：交换律（a+b=b+a），结合律
- **减法**：注意负数与减法的交互（-(-a) = +a）
- **乘法**：交换律，分配律（a×(b+c) = a×b + a×c）
- **除法**：除以分数 = 乘以其倒数（a÷b/c = a×c/b）

**运算顺序（BODMAS/BIDMAS）：**
括号 → 指数 → 除/乘 → 加/减

**关键规则：**
> 正负数存在于数轴两侧；掌握运算优先级；乘除在加减之前。

### 第4章：Fractions, Percentages and Decimals（分数、百分数、小数）

**三种形式互转：**

| 形式 | 转换规则 |
|------|---------|
| 分数 → 小数 | 分子 ÷ 分母（如 3/4 = 0.75） |
| 小数 → 百分数 | × 100（如 0.75 = 75%） |
| 百分数 → 分数 | ÷ 100 化简（如 75% = 3/4） |
| 分数 → 百分数 | 先转小数再×100 |

**分数运算：**
- 加减：通分（找公分母）
- 乘法：分子×分子 / 分母×分母
- 除法：第一个分数 × 第二个分数的倒数

**百分数应用：**
- 求百分之X：× (X/100)
- 百分比增减：原值 × (1 ± X%)

**关键规则：**
> 理解三种形式相互转换；求百分比时先转为小数再计算。

### 第5章：Equivalence, Inequalities, Units & Approximations（等式、不等式、单位换算、近似值）

**等式与不等式：**
- 等式：两边相等（`=`），可以对两边做同样的运算
- 不等式：`>`, `<`, `≥`, `≤`
- 求解不等式与方程相似，但乘除负数时不等号方向翻转

**单位换算：**

| 类别 | 常用换算 |
|------|---------|
| 长度 | 1m = 100cm = 1000mm；1km = 1000m |
| 质量 | 1kg = 1000g；1t = 1000kg |
| 容量 | 1L = 1000mL |
| 时间 | 1h = 60min；1min = 60s |

**近似值（Approximations）：**
- 四舍五入到指定位数
- 上下文决定近似精度（如货币精确到分，距离可能精确到km）

**关键规则：**
> 不同测量场景用不同单位；近似值基于情境判断；乘除不等式时注意方向。

### 第6章：Shapes（图形）

**二维图形分类：**
- 多边形（polygon）：三角形、四边形、五边形…
- 特殊四边形：正方形、长方形、平行四边形、菱形、梯形
- 圆形（circle）

**三角形分类：**
- 等边三角形（3边相等）
- 等腰三角形（2边相等）
- 不等边三角形（3边不等）
- 直角三角形（含一个90°角）

**多边形内角和：**
- n边形内角和 = (n-2) × 180°
- 三角形：180°；四边形：360°；五边形：540°

**关键规则：**
> 熟悉各类图形的名称、属性及内角和公式。

### 第7章：Area, Perimeter and Volume（面积、周长、体积）

**周长公式：**
- 正方形：4s
- 长方形：2(l+w)
- 圆形（circumference）：2πr 或 πd

**面积公式：**
- 正方形：s²
- 长方形：l × w
- 三角形：½ × b × h
- 平行四边形：b × h
- 梯形：½ × (a+b) × h
- 圆形：πr²

**体积公式：**
- 正方体：s³
- 长方体：l × w × h
- 圆柱体：πr²h

**关键规则：**
> 记忆常用面积/体积公式；注意单位（面积用²，体积用³）；π ≈ 3.14 或使用计算器值。

### 第8章：Symmetry & Shape Patterns（对称与图形规律）

**对称类型：**
- **线对称（反射对称）**：沿对称轴折叠后两半完全重合
- **旋转对称**：旋转一定角度后与原图形重合（旋转对称阶数）
- 正方形有4条对称轴，等边三角形有3条

**图形规律（Shape Patterns）：**
- 识别图形序列中的重复或变换规律
- 关注：形状、颜色、大小、数量、位置的变化
- 可能是抽象推理题型的前身

**平移/旋转/反射：**
- 平移（Translation）：位置移动，形状不变
- 旋转（Rotation）：绕固定点旋转
- 反射（Reflection）：沿轴翻转

**关键规则：**
> 熟悉对称轴、旋转对称阶数；识别图形变换类型（平移/旋转/反射）。

---

### 第9章：Angles（角度）

**角度类型：**
- 锐角（Acute）：< 90°
- 直角（Right）：= 90°
- 钝角（Obtuse）：90° < θ < 180°
- 平角（Straight）：= 180°
- 优角（Reflex）：> 180°

**重要角度关系：**
- 互补角（Supplementary）：两角之和 = 180°
- 互余角（Complementary）：两角之和 = 90°
- 对顶角（Vertically Opposite）：相等
- 同位角（Corresponding）：平行线中相等
- 内错角（Alternate）：平行线中相等

**三角形角度：**
- 内角和 = 180°
- 等边三角形每角 = 60°

**关键规则：**
> 记忆各角度名称和范围；掌握平行线角度关系；三角形内角和180°。

### 第10章：Reading Graphs（读图表）

**常见图表类型：**
- 条形图（Bar Chart）
- 折线图（Line Graph）
- 饼图（Pie Chart）
- 散点图（Scatter Plot）
- 直方图（Histogram）

**读图要点：**
- 识别坐标轴标签和单位
- 理解图例（legend）
- 从图中提取数值并进行计算（求总和、差、比例等）
- 识别趋势（上升/下降/稳定）

**饼图特点：**
- 整圆 = 100% = 360°
- 某扇形角度/360° = 该类别占比

**关键规则：**
> 先看标题、轴标签、图例；再读数值；注意单位；饼图角度=比例×360°。

### 第11章：Financial Maths - Savings & Profit（财务数学）

**核心概念：**
- **利润（Profit）**：收入 > 支出 → 剩余金额为利润
  - 例：售出$100，成本$50 → 利润 $50
- **亏损（Loss）**：收入 < 支出 → 差额为亏损
  - 例：付工资$50，收入$30 → 亏损$20
- **储蓄（Savings）**：原本要花出去的钱 > 实际花的钱 → 差额为储蓄

**折扣计算：**
- 折扣后价格 = 原价 × (1 - 折扣率)
- 例：$200打8折 = $200 × 0.8 = $160

**GST（商品服务税）= 10%：**
- 含税价 = 不含税价 × 1.1
- 不含税价 = 含税价 ÷ 1.1

**关键规则：**
> 利润 = 收入 - 支出；储蓄 = 原价 - 实付；熟悉折扣和税率计算。

### 第12章：Time & Date Calculations（时间与日期计算）

**时间换算：**
- 60秒 = 1分钟；60分钟 = 1小时；24小时 = 1天
- 7天 = 1周；约30/31天 = 1个月；365天 = 1年（闰年366天）

**12小时 vs 24小时制：**
- 下午1点 = 13:00；下午12点 = 12:00；午夜12点 = 00:00

**时间差计算：**
- 从较早时间到较晚时间的差值
- 跨越午夜时要特别注意进位

**日期计算：**
- 计算两个日期之间的天数
- 识别月份天数（1/3/5/7/8/10/12月31天；4/6/9/11月30天；2月28/29天）

**关键规则：**
> 注意12小时与24小时制的转换；跨越整点/午夜时加减进位。

### 第13章：Ratios and Speed（比率与速度）

**比率（Ratio）：**
- 表示两个量的相对关系（如 2:3 表示第一量是第二量的 2/3）
- 化简：找最大公因数（GCF）
- 比率分配：将总量按比率分配
  - 例：将$100按 2:3 分 → 2份和3份，各份 = $100÷5 = $20 → $40和$60

**速度公式：**
- **速度 = 距离 ÷ 时间**（S = D/T）
- **距离 = 速度 × 时间**（D = S × T）
- **时间 = 距离 ÷ 速度**（T = D/S）

**单位换算：**
- km/h ↔ m/s：× 1000/3600 = × 5/18（反向× 18/5）

**关键规则：**
> S=D/T 三角形记忆法；比率化简找GCF；按比率分配时先求单份。

### 第14章：Distance & Spatial Calculations（距离与空间计算）

**勾股定理（Pythagoras' Theorem）：**
- 直角三角形：a² + b² = c²（c为斜边）
- 用于求两点之间的直线距离

**坐标系（Coordinate System）：**
- 平面坐标（x, y）
- 两点距离公式：√[(x₂-x₁)² + (y₂-y₁)²]

**地图比例（Scale）：**
- 比例尺 1:50000 → 地图1cm = 实际500m
- 实际距离 = 地图距离 × 比例因子

**空间推理：**
- 从2D展开图识别3D图形
- 俯视图/正视图/侧视图的对应关系

**关键规则：**
> 勾股定理求斜边；地图比例尺换算；从展开图识别立体图形。

### 第15章：Chance & Probability（概率）

**概率基础：**
- 概率范围：0（不可能）到 1（必然发生）
- P(事件) = 有利结果数 ÷ 总可能结果数

**常见概率计算：**
- 掷骰子得6的概率：1/6
- 从52张牌中抽到红心：13/52 = 1/4

**互补事件：**
- P(事件发生) + P(事件不发生) = 1
- P(不发生) = 1 - P(发生)

**独立事件（连续）：**
- 两个独立事件同时发生：P(A) × P(B)

**树状图（Tree Diagram）：**
- 系统列举所有可能结果
- 每个分支的概率相乘，再相加求总概率

**关键规则：**
> 概率 = 有利/总数；互补 P + Q = 1；独立事件相乘；树状图系统列举。

### 第16章：Algebra & Expressions（代数与表达式）

**代数基础：**
- 字母代表未知数（变量/variable）
- 代数是"没有配料表的食谱"——给出步骤，你来找出未知量

**代数运算规则：**
- **加法**：同类项合并。2x + 5x + 2 = 7x + 2；x² + 4x + x = x² + 5x
- **减法**：同类项相减。5x - 2x + 1 = 3x + 1
- **乘法**：合并。5 · x · 6x · 2y = 60x²y（注：用"·"代替"×"避免与变量x混淆）
- **除法**：2x/4x = 1/2；2x/5y（不同变量无法化简）

**代入法（Substitution）：**
- 将数值代入变量求解
- 例：x = 3 时，3x² + 2x = 3(9) + 2(3) = 27 + 6 = 33

**BODMAS/BIDMAS 运算顺序适用于代数：**
> 先处理括号，再处理指数，再乘除，最后加减

### 第17章：Venn Diagrams & Transposing Worded Questions（维恩图与文字题转化）

**文字题转化（英语→数学）：**

| 英文表达 | 数学符号 |
|---------|---------|
| sum, plus, added, combined, increased by | + |
| minus, less, difference, decreased by | - |
| over, divided by, quotient, out of | ÷ |
| product, multiplied by, twice, thrice | × |
| equals, the same as, totals, equivalent to | = |

**未知数表示：** x, y, z 或 "?"

**解题策略：**
1. 将文字转化为数学公式
2. 代入选项答案验证（逆向法）
3. 求解方程

**维恩图（Venn Diagrams）：**
- 用重叠圆圈表示集合关系
- 交集（∩）：两个集合共有的元素
- 并集（∪）：两个集合所有的元素
- 互斥集合：无重叠部分

**维恩图应用题：**
- 通过已知总数、各集合数和交集数，求某个未知量
- 公式：|A ∪ B| = |A| + |B| - |A ∩ B|
